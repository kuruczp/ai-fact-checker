Download the extension

2

Unzip the file

Extract the zip to a permanent folder on your computer (e.g. your Desktop). Don't delete this folder.

3

Open Extensions

4

1

Download & unzip to your USB drive

Download ai-fact-checker.zip and extract it to a folder on your USB drive (e.g. E:\ai-fact-checker\).

2

Plug the USB into any computer

The extension folder travels with the drive — no internet download needed on each machine.

3

Open Extensions & enable Developer mode

In Chrome/Edge go to chrome://extensions or edge://extensions and turn on Developer mode.

4

Load unpacked from the USB

Click Load unpacked and navigate to the folder on your USB drive.

5

Keep the USB plugged in while browsing

Chrome links to the folder on the drive — the extension works as long as the USB is connected. Firefox users can install via about:addons and then unplug the drive.

💡 Tip: On Firefox, installing from file copies the extension internally — you only need the USB for the initial install, not while browsing.

Privacy & Security

Here's exactly what happens to your data.

📝

Only selected text is sent

Only the text you manually select is transmitted. The extension never reads the full page or your browsing history.

🔐

No API key in the extension

All AI calls go through a secure proxy server — no credentials are ever bundled in the extension or stored in your browser.

🚫

No tracking, no accounts

No analytics, no user identifiers, no personal data collected. No sign-up required.
