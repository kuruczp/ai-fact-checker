Checking: "The Great Wall of China is visible from space with the naked eye."
Partially True Confidence: HIGH
Summary

While the Great Wall is an impressive structure, it is generally not visible from space with the naked eye. The wall is wide enough, but it lacks sufficient contrast against the surrounding landscape at orbital distances.

Key Facts
  • The claim has been repeated since the 1930s but lacks evidence
  • NASA astronauts and Chinese astronaut Yang Liwei have confirmed it cannot be seen unaided
  • Width (~5–8 m) is comparable to a human hair at arm's length from orbit

Install the extension

Choose your browser below for step-by-step instructions.

1

Download the extension

Click the button below to download ai-fact-checker.zip.

2

Unzip the file

Extract the zip to a folder on your computer (e.g. your Desktop).

3

Open Extensions page

In Chrome, go to chrome://extensions and enable Developer mode (top-right toggle).

4

Load the extension

Click Load unpacked and select the unzipped folder. The extension icon appears in your toolbar.

5

You're done!

Select any text on any page → right-click → "Fact Check with AI".

Privacy & Security

We take your privacy seriously. Here's exactly what happens to your data.

📝

Only selected text is sent

Only the text you manually select is ever transmitted. The extension never reads the full page or your browsing history.

🔐

API key is server-side

The Anthropic API key lives on a secure proxy server — it is never bundled in the extension or stored in your browser.

🚫

No tracking, no accounts

We don't collect analytics, user identifiers, or any personal information. No sign-up required.

🌍

Open source

All extension code is open for inspection. If you find a security issue, please report it on GitHub.
