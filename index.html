Install the extension

Choose your browser for step-by-step instructions.

1

Download the extension

Click the button below to download ai-fact-checker.zip.

2

Unzip the file

Extract the zip to a permanent folder on your computer (e.g. your Desktop).

3

Open Extensions

Go to chrome://extensions and enable Developer mode (top-right toggle).

4

Load unpacked

Click Load unpacked and select the unzipped folder. The icon appears in your toolbar.

5

Done! Start using it

Select any text → right-click → "Fact Check with AI".

Privacy & Security

Here's exactly what happens to your data.

📝

Only selected text is sent

Only the text you manually select is transmitted. The extension never reads the full page or your browsing history.

🔐

No API key in the extension

All AI calls go through a secure proxy server — no credentials are ever bundled in the extension or stored in your browser.

🚫

No tracking, no accounts

No analytics, no user identifiers, no personal data collected. No sign-up required.

🌍

Open source

All extension code is open for inspection on GitHub.
